# Moise NIYOMAHORO Portfolio

A modern, responsive, and SEO-optimized portfolio website for Moise NIYOMAHORO, a Computer Systems & Network Engineer, Cybersecurity Specialist, and IoT Developer.

## Features
- Responsive HTML5 and CSS3 website
- Home, About, Projects, and Contact pages
- Dedicated pages for four GitHub projects
- GitHub Pages-ready structure
- Cybersecurity-inspired visual design

## Deployment to GitHub Pages
1. Create a GitHub repository named `my-portfolio`.
2. Push the project files to the repository.
3. In the repository settings, enable GitHub Pages and select the `GitHub Actions` source.
