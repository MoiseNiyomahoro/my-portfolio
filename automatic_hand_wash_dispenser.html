Project Detail

Automatic Hand Wash Dispenser

This project focuses on a touchless hand-wash system that improves hygiene and convenience by detecting user presence and automating soap or water flow.

Overview

The dispenser uses an ultrasonic or infrared sensor to detect a hand near the nozzle and trigger a controlled motor or pump action. The system is designed to be simple, low-power, and suitable for educational and small-scale deployment.

Technologies

ArduinoInfrared SensorServo/Motor DriverEmbedded C++

Components

  • Microcontroller board
  • Proximity sensor
  • Pump or actuator
  • Power supply and switching circuit

Implementation Details

Sensor input is read continuously, and the controller activates the actuator for a short interval when motion is detected. Timing, debounce logic, and safe power management are essential to avoid false triggers and excessive usage.

Code Sample

if (sensor.detected()) {
  activatePump();
  delay(1500);
  stopPump();
}
